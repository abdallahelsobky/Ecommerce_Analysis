# E-commerce Sales Analysis

**Tools:** Python, Pandas, Matplotlib  

**Description:**  
End-to-end analysis of e-commerce sales data, including data cleaning, aggregation, and visualization.

**Features:**  
- Cleaned and preprocessed raw data (handled missing values, type conversions)  
- Calculated total sales, top products, and revenue trends  
- Visualized daily, monthly, and category sales
